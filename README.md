# 基于STM32和人体节律的照明系统设计

## 项目概述

本项目是一个基于STM32G431RBTx微控制器的智能照明系统，通过人体节律调节灯光，提供更符合人体生理需求的照明环境。系统结合了光强传感器、实时时钟、RGB LED等组件，实现了智能调光、节律照明等功能。

## 硬件要求

- STM32G431RBTx开发板
- BH1750光强传感器
- WS2812 RGB LED
- LCD显示屏
- 按钮
- 其他必要的电路元件

### 系统架构

1. **核心层**：基于STM32 HAL库，负责硬件初始化和基本外设控制
2. **驱动层**：自定义驱动，包括GPIO、ADC、RTC、BH1750等
3. **应用层**：实现具体功能，如灯光控制、显示、用户交互等
4. **调度系统**：基于时间片的任务调度器，管理各个任务的执行

## 主要功能

### 1. 人体节律照明

根据时间自动调节灯光的亮度和色温，模拟自然光的变化，符合人体生理节律。

### 2. 环境光检测

使用BH1750光强传感器检测环境光强度，自动调整灯光亮度，保持舒适的光照环境。

### 3. 智能调光

通过PWM控制灯光亮度，实现平滑的亮度调节。

### 4. 实时时钟

使用RTC模块实现实时时间显示和时间相关的功能。

### 5. 用户交互

通过按钮和LCD显示屏实现用户交互，可手动调整灯光模式和参数。

### 6. RGB彩色灯光

使用WS2812 RGB LED实现彩色灯光效果，可根据不同场景选择不同的灯光模式。

## 系统流程

1. **系统初始化**：初始化各个外设和模块
2. **任务调度**：通过调度器管理各个任务的执行
3. **数据采集**：采集环境光强、时间等数据
4. **灯光控制**：根据采集的数据和用户设置，控制灯光的亮度、色温和颜色
5. **用户交互**：处理用户输入，更新显示信息

## 主要任务

| 任务名称               | 执行周期   | 功能描述      |
| ------------------ | ------ | --------- |
| task\_led\_proc    | 5ms    | LED状态处理   |
| button\_ticks      | 5ms    | 按钮状态检测    |
| task\_adc\_proc    | 20ms   | ADC数据处理   |
| task\_rtc\_proc    | 500ms  | 实时时钟处理    |
| task\_lcd\_proc    | 100ms  | LCD显示更新   |
| task\_bh1750\_proc | 200ms  | 光强传感器数据处理 |
| task\_lamp\_proc   | 1000ms | 灯光控制处理    |

## 开发环境

1. 使用STM32CubeMX打开`005_Calendar.ioc`文件，配置项目
2. 生成代码并在Keil MDK中打开项目
3. 编译项目生成hex文件

## 功能演示

### 正常模式

系统根据时间自动调节灯光，模拟自然光的变化：

- 早晨：偏蓝色调，亮度逐渐增加
- 中午：明亮的白色光
- 傍晚：偏黄色调，亮度逐渐减少
- 夜晚：低亮度的暖色调

### 手动模式

通过按钮切换到手动模式，可手动调整灯光的亮度和颜色。

### 场景模式

提供多种预设场景模式，如阅读、工作、休息等，一键切换到适合的灯光环境。

## 技术特点

1. **低功耗设计**：合理使用STM32的低功耗模式，延长系统运行时间
2. **模块化设计**：代码结构清晰，易于维护和扩展
3. **实时响应**：任务调度器确保系统的实时响应能力
4. **精确的时间控制**：使用RTC模块实现精确的时间管理
5. **智能算法**：根据环境光强和时间自动调整灯光参数
